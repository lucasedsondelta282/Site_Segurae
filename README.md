# SEGURAE - Plataforma de Encontros Simbióticos

## Sobre o Projeto
Plataforma web para conexão entre guardiões afetivos e pessoas que precisam de apoio emocional.

## Tecnologias Utilizadas
- HTML5
- CSS3
- JavaScript
- Font Awesome (ícones)
- Google Fonts (Poppins e Open Sans)
